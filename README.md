## 👩🏻‍💻 About

Personal portfolio site for Elizabeth Lovero built using Gatsby.

## 🚀 Quick start

1.  **Start developing.**

    ```sh
    gatsby develop
    ```

1.  **Open the source code and start editing!**

    The site is now running at `http://localhost:8000`!
